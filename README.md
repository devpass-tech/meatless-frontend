## Running locally

Clone the project
```bash
  git clone https://github.com/devpass-tech/project-meatless-react-template
```

Enter the project directory
```bash
  cd meatless-frontend
```

install the dependencies
```bash
  yarn install
```

start the server and enjoy :)
```bash
  yarn start
```
